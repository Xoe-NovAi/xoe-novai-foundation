The primary model for Ancient Greek on Hugging Face is **`pranaydeeps/Ancient-Greek-BERT`**.[1][2][3] Below are the specific details regarding its identification, architecture, and vector size.

### **Model Identification**
*   **Model ID:** `pranaydeeps/Ancient-Greek-BERT`[1][2][3][4]
*   **Alternative (Uncased):** `altsoph/bert-base-ancientgreek-uncased` (A fine-tuned version of the AUEB Greek BERT specifically for Ancient Greek texts).[2]

### **Architecture & Vector Size**[2]
*   **Vector Size (Hidden Dimension):** **768**[5][2][1]
    *   This is a standard `bert-base` architecture.[1][2]
*   **Parameters:** ~110M - 113M.[2][3]
*   **Layers:** 12 layers.[2][1]
*   **Attention Heads:** 12 heads.[2]
*   **Max Sequence Length:** 512 tokens.[4][2][5][1]

### **Technical Details**
*   **Base Model:** It was initialized from the **AUEB NLP Group's Greek BERT** (`nlpaueb/bert-base-greek-uncased-v1`) and subsequently trained on Ancient and Medieval Greek corpora.[2][4]
*   **Training Data:** Monolingual data from the First1KGreek Project, Perseus Digital Library, PROIEL Treebank, and Gorman's Treebank.[4][2][1]
*   **Preprocessing:** The model is **uncased** and utilizes standard de-accentuating (removing diacritics) for Greek text.[2]
*   **Performance:** Achieves state-of-the-art results for Part-of-Speech (PoS) tagging and Morphological Analysis on Ancient Greek treebanks (averaging >90% accuracy).

### **Usage Example (Python)**
```python
from transformers import AutoTokenizer, AutoModel[4][5][2][1]

# Load the model and tokenizer
model_id = "pranaydeeps/Ancient-Greek-BERT"
tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModel.from_pretrained(model_id)[2][1][4]

# Example text (Ancient Greek)
text = "μῆνιν ἄειδε θεὰ Πηληϊάδεω Ἀχιλῆος"
inputs = tokenizer(text, return_tensors="pt")
outputs = model(**inputs)[2]

# The last hidden state has the vector size of 768
embeddings = outputs.last_hidden_state
print(embeddings.shape)  # torch.Size([1, sequence_length, 768])
```[2]
