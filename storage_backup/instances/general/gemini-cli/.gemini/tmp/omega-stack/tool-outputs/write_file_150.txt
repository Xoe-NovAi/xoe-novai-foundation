## ⚠️ Risks & Mitigations
- **Context Hallucination**: Librarian might summarize falsely. *Mitigation*: Multi-turn verification where a model "reviews" the Librarian's summary against raw history.
- **Resource Contention**: Background worker stealing CPU. *Mitigation*: Run with `nice` priority and limit its thread count via `LLAMA_CPP_N_THREADS`.
