# Antigravity Manager

I can help you manage the Antigravity system.

Usage:
- `!./scripts/antigravity-maintenance.sh status` : Check account health
- `!./scripts/antigravity-maintenance.sh sync` : Sync credentials to instances
- `!./scripts/antigravity-maintenance.sh cleanup` : Clean logs
- `!node scripts/antigravity-direct-login.js` : Start sovereign login flow (Interactive)

To perform an action, just tell me which one you want.
