# Glossary: Vikunja Integration Grok MC

- Vikunja: Kanban/PM platform used for task tracking in the project.
- Memory Bank: Central memory store for cross-agent context and facts.
- EKB: Expert Knowledge Base; curated knowledge assets used by the agents.
- Ma'at: The governance/ethics framework guiding decisions and transparency.
- Sovereign Security Trinity: Security posture including rootless containers, SBOMs, scans.
- MCP: Memory Control Protocol; tools and configurations for MCP in this stack.
- Rootless Podman: Podman running without root privileges for security.
- Runbook: Step-by-step operational procedure.
- Runbook non-interactive: Procedure executable without human prompts.
- Import: Migrating tasks from memory_bank to Vikunja.
