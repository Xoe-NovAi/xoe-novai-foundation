<section><h1>Qdrant Documentation</h1>
<p>Qdrant is an AI-native vector database and a semantic search engine. You can use it to extract meaningful information from unstructured data.</p>
<p><a href="https://github.com/qdrant/qdrant_demo/">Clone this repo now</a> and build a search engine in five minutes.</p>

<a href="https://qdrant.tech/documentation/quickstart-cloud/">Cloud Quickstart

</div></section><section><h2>Ready to start developing?</h2>
<p>Qdrant is open-source and can be self-hosted. However, the quickest way to get started is with our <a href="https://qdrant.to/cloud">free tier</a> on Qdrant Cloud. It scales easily and provides a UI where you can interact with data.</p>
