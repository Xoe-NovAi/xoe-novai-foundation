# Grok Glossary - Vikunja Sovereign Migration (v1.0.0)

- Vikunja: Kanban/PM tool for task management.
- Memory Bank: Long-term memory store for agents and context.
- EKB: Expert Knowledge Base; knowledge assets used for reasoning and guidance.
- Ma'at: Integrity/governance framework; zero-telemetry and sovereignty guardrails.
- Rootless Podman: Running Podman as a non-root user for security.
- Secrets lifecycle: Pre-create secrets via Podman secrets; rootless compose enforces truth.
- Runbook: Step-by-step operational procedure for spin-ups.
- Importer: Tool that migrates memory_bank data into Vikunja.
- Namespace/Project: Vikunja structure of the workspace; mapping from memory_bank projects.
- Dry-run: Non-destructive test to simulate actions.
