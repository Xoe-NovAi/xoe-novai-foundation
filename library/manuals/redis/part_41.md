Release 01
## Redis for AI—for faster GenAI apps
Redis for AI is an integrated package of our offerings that get GenAI apps into production faster.
Watch full video
  * The fastest vector database
Build with a real-time architecture for GenAI apps.
  * LLM memory
Build the fastest, most reliable GenAI apps with our advanced vector database.
  * Agentic memory
Make agents faster for more complex reasoning and better answers.
  * Semantic caching
Retrieve stored answers fast and save >30% on costly LLM calls.
  * Feature store
Get production ML models predictions in less than 1 millisecond.


### Release 02: Cache 5X more for the same price with Redis Flex
We’ve reduced the cost of deployments by up to 80 percent and made apps faster— without changing any code. Now a first-party service, Auto Tiering supports more use cases and customers across the world.
[See how you can save](https://redis.io/meeting/)
RELEASE 03
## Introducing Redis 8
Redis 8 brings the most advanced Redis features to every single Redis user, starting with Community Edition.
Watch full video
  * Store, update, & retrieve JSON data
Enable fast, responsive operations and dynamic interactions.
  * Build smarter, faster GenAI apps
Boost query throughput with our vector database.
  * Search & query data faster
Work through millions of records quickly to transform data into action.
  * Optimize time series data
Maintain high throughput and low latency for real-time insights.
  * Simplify probabilistic data
Store probabilities to efficiently represent underlying phenomena.


RELEASE 04
## Work faster with Redis Copilot
Our free virtual AI assistant, Redis Copilot, is here. Use Copilot in Redis Insight to answer questions, get data insights, and more.
Watch full video
  * Docs content
Get how-to content on everything from formatting queries to setting up a cluster.
  * Coding support
Tell Redis Copilot what you need and get code snippets or commands that do just that.
  * Ask questions
Redis Copilot can write queries to use on your data to answer key questions.


Release 05
## Redis Data Integration (RDI) in Redis Cloud
Sync data in existing databases with Redis Cloud automatically. We’ve done the heavy lifting, so your apps can run faster without the costs of building data pipelines.
Watch full video
  * Federate your data with automated pipelines
Interact with all your data in one unified API.
  * Simplify development
Use pre-existing templates to get set up and going fast with minimal rework.
  * Accelerate apps
Modernize your apps for fast performance without redesigning your data strategy.
