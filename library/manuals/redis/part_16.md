## Frequently asked questions
[More questions? See our Docs page](https://redis.io/docs/latest/)
What is Redis Cloud?
Redis Cloud is the fully-managed version of the fastest and most feature-rich Redis on the market. By choosing our Redis as a service, you can easily start putting Redis 8 to use in production and scale real-time speeds effortlessly on a next generation architecture that goes beyond what is possible with Redis Open Source.
How can you use Redis Cloud?
It’s easy to get started. Launch a free Redis Cloud database at <https://redis.io/try-free/>.
What’s the difference between Redis Cloud and Redis Software?
Redis Cloud and Redis Software are built on similar next generation architectures that go beyond what is possible with Redis Open Source. Redis Cloud is the fully managed version of Redis while Redis Software is for on-premise environments.
What’s the difference between Redis Cloud and Redis Open Source?
Redis Cloud goes beyond Redis Open Source with enterprise-grade architecture. It includes a cluster manager & proxy, faster speeds, higher availability (99.999% availability with Active-Active), and advanced capabilities including Redis Flex.
What are the benefits of using Redis Cloud?
If you’re on AWS or GCP, Redis Cloud is the only fully managed service that offers Redis 8. Other providers are at least a generation behind and limited to Redis 7.2 or older.
If you’re on Azure, we’ve worked with Microsoft to create [Azure Managed Redis](https://azure.microsoft.com/en-us/products/managed-redis), a first party Azure service with Azure support built on Redis Cloud technology.
What can you build with Redis Cloud?
Redis Cloud supports a wide range of use cases, including caching, session management, document database, search engine, semantic caching, agent memory, vector database, semantic routing, rate limiting, and pub/sub messaging.
