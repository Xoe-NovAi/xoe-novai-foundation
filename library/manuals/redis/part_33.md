## Frequently asked questions
[More questions? See our Docs page](https://redis.io/docs/latest/)
What is Redis Insight?
Redis Insight is the perfect developer tool for anyone using Redis, whether it's Redis Software, Redis Cloud, Redis Community Edition, Azure Managed Redis, Redis on AWS, Redis on GCP, or other Redis-compatible products. It speeds your app development lifecycles by making it easier to develop, debug, and manage your Redis data. Best of all, it’s 100% free and available on desktop and the Redis Cloud console.
What are the key benefits of using Redis Insight?
Redis Insight offers a variety of advanced capabilities designed to simplify working with Redis databases. With an intuitive and user-friendly graphical user interface (GUI) and advanced Command Line Interface (CLI), you can develop and debug with ease.
This experience is complemented by features such as our schema-aware auto-complete, command profiling, data visualization, and slow logs. You can even utilize our AI-powered chatbot, Redis Copilot, to learn more about your data or even build queries with natural language.
How do you get started with Redis Insight?
There are a few ways to access Redis Insight. You can:
• Open your database in Redis Insight via the Redis Cloud console (currently available for the Redis Essentials tier)
• Download it directly on your Windows, macOS, or Linux desktop client
• Run the Redis Insight Docker image
• Install on Kubernetes using a YAML file
• Install on an AWS EC2 instance to oversee AWS ElastiCache Redis deployments
[Get started in a few clicks](https://redis.io/docs/latest/operate/redisinsight/install/)
How can Redis integrate with my existing workflows?
There are a few ways to seamlessly extend the power of Redis into your development cycles. You can use Redis Data Integration (RDI) to establish a data streaming pipeline that copies data from your existing database to Redis. This makes it easy to create, test, deploy, and monitor your data pipelines directly in Redis Insight.
Additionally, with our new Redis for Visual Studio Code (VS Code) plugin, you can browse and interact with your data in real time, get access to an enhanced Redis CLI , and build queries faster—without leaving your Microsoft VS Code editor.
