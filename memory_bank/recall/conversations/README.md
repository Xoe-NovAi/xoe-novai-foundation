# Recall - Conversations

Session logs and conversation history.

## Naming Convention

```
YYYY-MM-DD_session-id.session.md
```

## Content Template

```markdown
---
type: episodic
session_id: ses_xxx
date: YYYY-MM-DD
duration: XX minutes
participants: [agent1, agent2]
topics: [topic1, topic2]
---

# Session: [Title]

## Summary
[Brief summary]

## Key Decisions
- [Decision 1]
- [Decision 2]

## Actions Taken
- [Action 1]
- [Action 2]

## Follow-ups
- [ ] [Follow-up 1]
```

## Retention

90 days. Older sessions are summarized and moved to archival.
