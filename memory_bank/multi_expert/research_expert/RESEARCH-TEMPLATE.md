# Research Job Template

Use this template when recording or proposing a new research job. Agents loading the Research Expert Memory Bank can reference these fields to guide their investigation.

```
### Job ID: RJ-xxx

**Title**: Short descriptive name

**Description**:
- What needs to be investigated or evaluated
- Context about why it's important

**Priority**: P1 / P2 / P3 / P4

**Estimated Effort**: X hours

**Relevant Files/Links**:
- link1
- link2

**Notes**:
- Current status or past findings
- Helpful hints or starting points

**Result Summary**:
- Findings
- Recommendations
- Next steps
```
